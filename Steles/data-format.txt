

Steles data-file spec

	File locations:
		/n.dat: datafile with song's number
		/n.tmp: currently-loaded temp-data file

	File contents:
		(393265 total bytes)
		Header bytes:
			(49 total bytes)
			0: BPM value
			1-48:
				Each sequence's size, in beats
				(* 24 * 4 = size in ticks)
		Body bytes:
			48 sequences (393216 total bytes)
			Per sequence (for 48 sequences):
				1024 tick-slots (8192 total bytes)
				Per tick-slot:
					2 notes (8 total bytes)
					Per note:
						(4 total bytes)
						0: Duration
						1: Channel (+ 144)
						2: Pitch
						3: Velocity
