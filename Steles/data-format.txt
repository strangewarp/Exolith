

Steles data-file spec

	File locations:
		/n.dat: datafile with song's number
		/n.tmp: currently-loaded temp-data file

	File contents:
		(294961 total bytes)
		Header bytes:
			(49 total bytes)
			0: BPM value
			1-48:
				Each sequence's size, in beats
				(* 24 * 4 = size in ticks)
		Body bytes:
			48 sequences (294912 total bytes)
			Per sequence (for 48 sequences):
				1024 tick-slots (6144 total bytes)
				Per tick-slot:
					2 notes (6 total bytes)
					Per note:
						(3 total bytes)
						0: Chan-dur composite
						1: Pitch
						2: Velocity
