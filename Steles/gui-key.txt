
Buttons (PLAY MODE):

	c s s s s
	c s s s s
	c s s s s
	c s s s s
	c s s s s
	c s s s s

	c: Control button
		* Hold one or several to toggle a control-command
	s: Sequence button
		* Applies various commands to a sequence


Buttons (RECORD MODE):

	c n R D U
	c n n n n
	c n n n n
	c n n n n
	c n n n n
	c n n n n

	c: Control button
		* Hold one or several to toggle the modification of var values
	n: Note button
		* Plays a note when no control-chord is held
		* Changes var values when a control-chord is held
	R: Random-interval button
		* When held: notes play a random interval from the channel's most recent note, or record the random-command into the sequence
	D: Down-interval button
		* When held: notes play a downward interval from the channel's most recent note, or record the generic-interval command into the sequence
	U: Up-interval button
		* When held: notes play an upward interval from the channel's most recent note, or record the generic-interval command into the sequence
