

Buttons (PLAY MODE):

	c s s s s
	c s s s s
	c s s s s
	c s s s s
	c s s s s
	c s s s s

	c: Control button
		* Hold one or several to toggle a control-command
	s: Sequence button
		* Applies various commands to a sequence


Buttons (RECORD MODE):

	c n n n n
	c n n n n
	c n n n n
	c n n n n
	c n n n n
	c n n n n

	c: Control button
		* Hold one or several to toggle the modification of var values
	n: Note button
		* Plays a note when no control-chord is held
		* Changes var values when a control-chord is held


LEDs (PLAY MODE):

	I  c c c c c c c c  O
	   a a a a a a a a

	   q q q q q q q q
	   q q q q q q q q

	P  q q q q q q q q  P

	I: Analog MIDI-IN indicator
	O: Analog MIDI-OUT indicator
	P: Currently active page

	a: Number of active sustains
	c: Global cue point
	q: Seq activity, or SCATTERed seqs when SCATTER is held
